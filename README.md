# Marathon Skills Web

Статическая веб-копия WPF-приложения Marathon Skills.

## Возможности

- главная страница с таймером до ближайшего марафона 15 июня;
- регистрация участников;
- добавление фотографии;
- расчет BMI с цветной шкалой и визуальным индикатором;
- просмотр, редактирование и удаление участников;
- сохранение данных в `localStorage`.

## Локальный запуск

Откройте `index.html` напрямую или запустите любой статический HTTP-сервер.

## Публикация

Проект не требует сборки и подходит для прямого размещения на Vercel.
