# Marathon Skills Web

Статическая веб-копия WPF-приложения Marathon Skills.

## Возможности

- главная страница с таймером до ближайшего марафона 15 июня;
- регистрация участников;
- добавление фотографии;
- проверка формата и размера фотографии перед добавлением;
- расчет BMI с цветной шкалой и визуальным индикатором;
- расчет основного обмена калорий с учетом пола и возраста;
- просмотр, редактирование и удаление участников;
- динамический поиск, фильтрация и сортировка списка участников;
- сохранение данных в `localStorage`.

## Архитектура

- `domain.js` содержит бизнес-логику без обращений к DOM: состояние приложения, коллекцию участников, валидацию фото, BMI, калории и таймер.
- `app.js` отвечает за отображение данных и обработку действий пользователя.
- `index.html` и `styles.css` описывают интерфейс приложения.

## Локальный запуск

Откройте `index.html` напрямую или запустите любой статический HTTP-сервер.

## Публикация

Проект не требует сборки и подходит для прямого размещения на Vercel.
